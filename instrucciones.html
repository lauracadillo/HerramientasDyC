Instrucciones

Cómo ejecutar los scripts

Seguir estos pasos cada vez que se quiera ejecutar uno de los notebooks.

01
Seleccionar el perfil correcto
Elegir su perfil (Laura / Julio / Lidia) en la página de Scripts antes de ejecutar. Esto ajusta automáticamente las rutas de archivo al usuario de Windows correspondiente.
02
Abrir el notebook desde la página "Herramientas"
Hacer clic en la tarjeta del script. Se abrirá directamente en VS Code. Si el navegador pide permiso, seleccionar "Abrir" y marcar "Recordar mi elección".
03
Actualizar las rutas de entrada si es necesario
En la primera celda del notebook se encuentran las variables de configuración. Hay que asegurarse de que los nombres de los archivos Excel fuente coincidan con los que tienes en tu carpeta.
# Primera celda — variables de configuración ARCHIVO_ENTRADA = r"C:\Users\{usuario}\OneDrive - Integratel Perú\Datos\archivo.xlsx" MES = "2025-05" # ← cambia solo esto si es necesario
Actualizar el mes con el actual para que los cálculos se realicen bien
04
Ejecuta todas las celdas
En VS Code usa el botón "Run All" en la parte superior del notebook.
05
Revisa el archivo de salida
Al finalizar, el notebook genera un Excel de salida en la misma carpeta del script.
El nombre del archivo de salida incluye la fecha de ejecución automáticamente.
