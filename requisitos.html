Requisitos

Pasos previos al funcionamiento de esta página

Requisitos para poder ejecutar los scripts de esta página sin problemas.

01
Instalar Visual Studio Code desde el centro de software
02
Instalar Python desde el centro de software
03
Abrir "Windows PowerShell"
04
Instalar paquetes necesarios
Copiar una por una las siguientes lineas en la consola para su ejecucion.
pip install pandas
pip install openpyxl
pip install numpy
pip install tabulate
pip install pywin32
04
Ejecuta todas las celdas
En VS Code usa el botón "Run All" en la parte superior del notebook.
